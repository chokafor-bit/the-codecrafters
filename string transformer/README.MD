# SENTINEL String Transformer

A Go-based command-line tool for text transformations and string analysis.

### How to Run
1. Install **Go**.
2. Save the code as `main.go`.
3. Run: `go run main.go`

### Command Examples

1. **UPPER**: `hello` -> `HELLO`
2. **lower**: `BYE` -> `bye`
3. **Title**: `the king of the hill` -> `The King of the Hill`
4. **Reverse**: `Go is fun` -> `oG si nuf`
5. **Snake**: `Hello World!` -> `hello_world`

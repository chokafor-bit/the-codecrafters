# CLI Calculator

A simple calculator that runs in the terminal. I built this while learning Go.

## How to run it

Make sure you have Go installed, then just run:

```bash
go run main.go
```

## How it works

It asks you for two numbers and an operator, then prints the answer. That's it.

```
Enter first number: 10
Enter operator (+, -, *, /): +
Enter second number: 5
Answer: 15
```

Supports +, -, *, and /. If you try to divide by zero it'll tell you off.

And i added help function and goodbye massege if the user exit the calculator.

## Why I built this

Just a beginner project to get comfortable with user input and switch statements in Go.
